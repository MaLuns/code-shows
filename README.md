# Code-Shows
Code-shows 是一个将代码片段生成为可在线编辑的静态网页的工具。

## 快速开始

**安装 Code-Shows**
```
npm install code-shows -g
```
or
```
yarn global add code-shows
```

**初始化项目**
```
ml-code init [name]
cd [name]
```

**启动本地服务**
```
ml-code serve
```

**创建代码片段**
```
ml-code new <name>
```

**生成静态文件**
```
ml-code generate
```
